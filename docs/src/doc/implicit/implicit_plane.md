# Implicit plane

## In 2D

Implicit function:

$$F(x,r) = |x|$$

Gradient:

$$
\begin{align}
\nabla_x F(x,r) &= \text{sgn}(x) \\
\nabla_r F(x,r) &= 0
\end{align}
$$

Hessian is zero.

## In 3D

Implicit function:

$$F(x,y,z) = |x|$$

Gradient:

$$
\begin{align}
\nabla_x F(x,y,z) &= \text{sgn}(x) \\
\nabla_y F(x,y,z) &= 0 \\
\nabla_z F(x,y,z) &= 0
\end{align}
$$

Hessian is zero.
